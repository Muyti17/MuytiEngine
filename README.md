# MUYTI Engine

MUYTI Engine is a modern, modular, high-performance game engine written in C++20.

The project is being developed from scratch with a strong focus on clean architecture, scalability, and full control over every major subsystem.

---

## Vision

Create a professional game engine that is powerful enough for large-scale games while remaining understandable and extensible.

---

## Goals

- Modern C++20
- Modular architecture
- Vulkan Renderer
- DirectX 12 Renderer
- Custom Rendering Hardware Interface (RHI)
- Custom GUI Framework (MuytiGUI)
- Custom Physics System
- Custom Audio System
- Built-in Editor
- Cross-platform support

---

## Current Status

🚧 Early Development

The project is currently in the architecture and core development stage.
